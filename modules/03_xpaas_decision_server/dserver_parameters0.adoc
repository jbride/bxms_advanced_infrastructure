== xPaaS Decision Server

.Parameters

* `SOURCE_REPOSITORY_URL`
** URL to version-controlled application source code
** Used during S2I

* `SOURCE_REPOSITORY_REF`
** Git branch/tag reference
** Default: Master
** Used during S2I

* `CONTEXT_DIR`
** Path within Git project to build
** Used during S2I
