== XPaaS DecisionServer Params

. *SOURCE_REPOSITORY_URL*
.. URL to version controlled app source code.
.. Utilized during S2I.

. *SOURCE_REPOSITORY_REF*
.. Git branch/tag reference. Default: _master_.
.. Utilized during S2I.

. *CONTEXT_DIR*
.. Path within the Git project to build.
.. Utilized during S2I.
